Here the user can add or edit FREEWORK spots which are automatically added to the database in realtime and added to the FREEWORK map

FREEWORK MAPâ€‹

The FREEWORK main map shows all nice FREEWORK spots in an area around your current location where different colours of markers indicate different kinds of spots. So far the colours and categories are as followed: - blue = public - orange = community - red = remote - pink = super remote - yellow = AIR FREEWORK spots which you can rent for a day

The map also enables a button to jump to your current position as well as buttons as links to GoogleMaps for the navigation, once you choosed your FREEWORK spot for the day.
